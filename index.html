HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Styling link in CSS needs to follow a specific order of link states: a > a:link > a:visited > a:focus > a:hover > a:active
  • CSS: Cascading Style Sheets

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: check the branch adding the changed files to the staging area
  • git commit -m: "added CSS to project": Commit the changes to repository with description message
  • git pull origin main: make sure current branch is up to date by pulling latest version of Main branch before opening a pull request
  • git push origin branch-name: push changes to Github remote server
  • Markdown Syntax cheat sheet

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
